Manual Completo de Ambiente: VCTP

Este guia detalha cada passo necessário para configurar o ambiente de Visão por Computador do zero.

1. Onde buscar as ferramentas

  • VS Code: code.visualstudio.com (Editor de código)
  • MSYS2: msys2.org (Gestor de compilador e bibliotecas)
  • Extensões VS Code: Instala o pack "C/C++ Extension Pack" da Microsoft.

2. Instalação do MSYS2 e Compilador

1. Instala o MSYS2 em C:\msys64.

2. Abre o terminal MSYS2 MinGW64 e instala o g++:

pacman -S mingw-w64-x86_64-gcc

3. Instalação do OpenCV

No mesmo terminal MSYS2 MinGW64, instala o OpenCV:

pacman -S mingw-w64-x86_64-opencv

*Este processo pode demorar alguns minutos dependendo da internet.

4. Variáveis de Ambiente (Passo Crítico)

Para o Windows encontrar o compilador e o OpenCV:

  1. Pesquisa por "Variáveis de Ambiente" no Windows.
  2. No Path do Sistema, clica em "Editar" e adiciona:
C:\msys64\mingw64\bin

Nota: Reinicia o VS Code depois de fazeres isto!

5. Estrutura de Pastas do Projeto

Garante que tens esta organização para evitar erros de caminho:

C:\VCTP\
├── main.cpp C++
├── video.avi Vídeo
└── lib\
    ├── vc.h Header
    └── vc.c Implementação C

6. Configuração VS Code (tasks.json)

Vai a Terminal > Configure Tasks e substitui os "args" por estes:

"args": [
  "-g", "${file}",
  "-x", "c", "${fileDirname}/lib/vc.c", "-x", "none",
  "-o", "${fileDirname}/main.exe",
  "-I", "C:/msys64/mingw64/include/opencv4",
  "-L", "C:/msys64/mingw64/lib",
  "-lopencv_core", "-lopencv_highgui", "-lopencv_imgproc", "-lopencv_videoio"
]

7. O Comando "Mestre" de Compilação

Se quiseres compilar manualmente no terminal do VS Code:

g++ -g C:\VCTP\main.cpp -x c C:\VCTP\lib\vc.c -x none -o C:\VCTP\main.exe
-I C:/msys64/mingw64/include/opencv4
-L C:/msys64/mingw64/lib
-lopencv_core -lopencv_highgui -lopencv_imgproc -lopencv_videoio
Guia VCTP Final - Configuração Completa e Testada
