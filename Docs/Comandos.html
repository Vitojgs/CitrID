Cheat Sheet: Git & Compilação

Comandos Git Essenciais

  • git status - Verifica o estado dos ficheiros.
  • git add . - Adiciona todas as alterações ao índice.
  • git commit -m "mensagem" - Regista as alterações com uma mensagem.
  • git push origin [branch] - Envia commits para o servidor.
  • git pull - Atualiza o repositório local.
  • git rm --cached [ficheiro] - Remove um ficheiro do Git sem apagar o ficheiro localmente.

Comandos de Compilação (g++)

Para testar a sua parte:

g++ main.cpp segmentacao.cpp lib/vc.c -o main -I C:/msys64/mingw64/include/opencv4 -I C:/msys64/mingw64/include/opencv4/opencv2 -L C:/msys64/mingw64/lib -lopencv_core -lopencv_highgui -lopencv_videoio -lopencv_imgproc -std=c++17 -lm

Para compilar com todos os módulos (Integração):

g++ main.cpp segmentacao.cpp blobs.cpp geometria.cpp tracking.cpp lib/vc.c -o main -I/usr/include/opencv4 -lopencv_core -lopencv_highgui -lopencv_videoio -lopencv_imgproc -std=c++17 -lm
